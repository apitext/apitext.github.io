how do I install apitext?

  1. Download the compressed apitext file folder here.
  2. Unzip the contents of the file.
  3. Modify the file name (remove the "-master" at the end of the file.
  4. Modify the contents of the "config.json" file.
  5. Using the FTP program of your choice, place the API folder inside the folder that you want it to control.
  6. Move the ".htaccess" file outside the API folder.
